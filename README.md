# 🚗 Car Website

A modern and responsive car showcase website built to display luxury, sports, and modern vehicles with a clean user interface and smooth user experience.

---

## ✨ Features

- 🔥 Modern and responsive UI
- 🚘 Car showcase section
- 📱 Mobile-friendly design
- ⚡ Fast and optimized performance
- 🎨 Clean animations and transitions
- 🔍 Search and filter functionality
- 📸 High-quality car images
- 🌙 Stylish layout and color palette

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
